🔄 Prediction Processing Flow

Quy Trình Chi Tiết

  1. User chọn ảnh X-ray → Upload
  2. Frontend validate file (type, size)
  3. Frontend send to API (POST /api/predict/upload)
  4. API verify JWT token
  5. API save file to disk
  6. API create prediction record (status=queued)
  7. API enqueue Celery task
  8. API return task_id to frontend
  9. Frontend open WebSocket for real-time updates
  10. Celery Worker dequeue task từ Redis queue
  11. Worker load 4 AI models (UNet, DenseNet, EfficientNet, DenseNet_T2)
  12. Worker preprocess image (resize, normalize)
  13. Worker run inference trên 4 models
  14. Worker generate Grad-CAM heatmaps
  15. Worker extract bounding box
  16. Worker ensemble predictions (majority voting)
  17. Worker classify disease type (Bacterial/Viral/COVID)
  18. Worker update prediction record (results + files paths)
  19. Worker send WebSocket event (status=done)
  20. Frontend receive notification → Display results
  21. User view results (prediction, heatmaps, confidence)
  22. User can add note → Doctor review
  23. User can export PDF report
🎯 Key Points:
• Real-time WebSocket updates cho progress tracking
• Celery + Redis cho async task processing
• 4 models ensemble cho high accuracy
• Grad-CAM heatmaps cho interpretability
• PDF export cho doctor review

🔐 Authentication Flow

Registration → Login → Token Refresh → Logout

🔑 Token Management:
• Access Token: 8 hours expiry → Short lived
• Refresh Token: 30 days expiry → Stored in DB
• Password: Hashed with bcrypt
• Rate Limiting: 10 requests/min cho login
• Account Locking: Sau 5 lần đăng nhập thất bại (30 min)

Luồng Đăng Nhập Chi Tiết

Step Description
1 User nhập username + password
2 Frontend validate, send to API
3 API check rate limit (10 req/min)
4 API query user by username
5 Nếu không tìm → Error 401
6 Verify password (bcrypt)
7 Nếu sai → Increment failed_count
8 Nếu failed_count >= 5 → Lock account 30min
9 Nếu đúng → Generate tokens (JWT)
10 Store refresh_token hash in DB
11 Create audit log (action=login)
12 Return tokens to frontend
13 Frontend save in localStorage
